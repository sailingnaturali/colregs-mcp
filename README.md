# colregs-mcp

A public MCP (Model Context Protocol) server that exposes the International Regulations for Preventing Collisions at Sea (COLREGs / COLREGS 72) as a queryable tool surface — returning rule text by number or keyword, determining the required lights and shapes for a given vessel type and condition, and checking a described scenario for compliance with the applicable rules. Designed for use with navigation agents and voyage-planning assistants.
